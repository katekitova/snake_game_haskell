# Snake Game

Аркадная игра «Змейка» на Haskell с рендерингом через `gloss`.
Проект выполнен для Практикума ВМК МГУ 2026.

## Функциональность

- Три режима: `Classic`, `Wrap`, `MultiFood`
- Меню выбора режима перед стартом
- Подсчёт текущего счёта и отображение рекорда
- Сохранение рекорда между запусками
- Пауза и быстрый перезапуск игры
- Экран окончания игры (`Game Over`)
- Пиксельные спрайты для яблока и сегментов змейки
- Фоновая музыка и звуковые эффекты

## Правила игры

- Змейка движется по сетке и управляется стрелками.
- При съедании яблока змейка растёт на один сегмент.
- При съедании яблока счёт увеличивается на `+1`.
- При съедании яблока игра немного ускоряется.
- Проигрыш наступает при столкновении с собственным телом.
- В режимах `Classic` и `MultiFood` проигрыш также наступает при столкновении со стеной.
- В режиме `Wrap` выход за границу поля переносит змейку на противоположную сторону.
- В `MultiFood` на поле одновременно несколько яблок (в текущей реализации — 3).

## Управление

- `Enter` — старт игры из меню
- `1` — выбрать режим `Classic`
- `2` — выбрать режим `Wrap`
- `3` — выбрать режим `MultiFood`
- `Стрелки` — движение
- `P` — пауза / продолжение
- `R` — перезапуск текущей игры
- `Esc` — возврат в меню (из игрового экрана)
- `Q` — выход из приложения

## Требования

- [Stack](https://www.haskellstack.org/)
- Графическое окружение с поддержкой OpenGL (для `gloss`)
- macOS для аудио (`afplay` и `pkill`)

## Сборка и запуск

```bash
stack setup
stack build
stack run
```

## Полезные команды

```bash
stack build --file-watch

stack ghci
```

## Данные и ресурсы

- Рекорд: `data/best_score.txt`
- Спрайты: `data/apple-pixel.bmp`, `data/snake_*.bmp`
- Звуки: `data/menu.wav`, `data/background.wav`, `data/chew.wav`, `data/death.wav`

## Структура проекта

- `app/Main.hs` — точка входа, загрузка ресурсов, запуск `playIO`
- `src/Types.hs` — основные типы состояния игры
- `src/Logic.hs` — игровая логика, столкновения, очки, рекорд, управление музыкой/звуками
- `src/Input.hs` — обработка клавиш
- `src/Render.hs` — отрисовка меню, поля, змейки, еды и интерфейсных элементов
- `test/Spec.hs` — тестовый модуль, не используется
- `data/` — игровые данные и медиа-файлы

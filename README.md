# Snake Game

Реализация игры "Змейка" на Haskell с рендерингом через Gloss для Практикума ВМК МГУ 2026.

## Возможности

- Три режима игры: `Classic`, `Wrap` и `MultiFood`
- Подсчёт текущего счёта
- Сохранение рекорда
- Пауза и рестарт
- Экран меню и экран окончания игры

## Управление

- `Enter` — начать игру из меню
- `1` — режим `Classic` (столкновение со стеной - проигрыш)
- `2` — режим `Wrap` (переход через края поля)
- `3` — режим `MultiFood` (`Classic` с несколькими объектами еды)
- `Стрелки` — движение
- `P` — пауза
- `R` — рестарт после `Game Over`
- `Esc` — выход в меню
- `Q` — выход из игры

## Сборка и запуск

Проект использует [Stack](https://www.haskellstack.org/).

```bash
stack setup
stack build
stack run
```

Собрать и запустить проект можно при помощи команды:

```
stack run
```

Чтобы проект автоматически пересобирался при обновлении исходных файлов, используйте команду:

```
stack build --file-watch
```

Чтобы запустить интепретатор GHCi и автоматически подгрузить все модули проекта, используйте команду:

```
stack ghci
```

## Структура проекта

- `src/` — библиотечная часть: игровая логика, рендеринг, типы, обработка ввода
- `app/` — исполняемый модуль (точка входа);
- `test/` — автоматические тесты проекта;
- `data/` — данные игры (например, файл с рекордом);
- `stack.yaml` — версия компилятора и зависимости;
- `package.yaml` — описание проекта, зависимости и опции компиляции;
- `README.md` — описание проекта, инструкции по сборке и запуску;

Остальные файлы — это дополнительные полезные настройки:

- `.gitignore`, указывающий Git не обращать внимания на временные файлы;
- `.ghci` для более чистого вывода интерпретатора GHCi.
